# Barbearia Dashboard (React)

Sistema demo com:
- UI moderna (tema escuro) + Sidebar
- Login demo (admin/admin)
- CRUD local (LocalStorage): barbeiros, serviços, atendimentos
- Cálculo automático de comissão
- Relatórios + exportação CSV
- Pronto para Deploy (Vercel)

## Rodar local
```bash
npm install
npm start
```

## Login
- usuário: **admin**
- senha: **admin**

## Observação
Os dados ficam no navegador (LocalStorage). No menu, use **Reset dados** para voltar ao padrão.
